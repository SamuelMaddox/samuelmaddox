## Story Book

Storybook is packaged as a small, development-only, workshop that lives alongside our app. It provides an isolated iframe to render components without interference from app business logic and context. Relevant documentation listed below:

- [Framework Integration](https://storybook.js.org/docs/get-started/frameworks)
- [Styling and CSS](https://storybook.js.org/docs/configure/styling-and-css)
- [How to Write Stories](https://storybook.js.org/docs/writing-stories)
- [Installing Addons](https://storybook.js.org/docs/addons/install-addons)
  - [A11Y](https://storybook.js.org/addons/@storybook/addon-a11y)
  - [Themes](https://storybook.js.org/addons/@storybook/addon-themes)
- [Storybook Autodocs](https://storybook.js.org/docs/writing-docs/autodocs?renderer=react)
- [Accessibility tests](https://storybook.js.org/docs/writing-tests/accessibility-testing)
- [Figma Plugin](https://storybook.js.org/docs/sharing/design-integrations?renderer=react#embed-storybook-in-figma-with-the-plugin)
