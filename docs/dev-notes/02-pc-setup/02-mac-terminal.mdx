# Mac Terminal

![PowerLevel10k](./assets/powerlevel-10k.png)

## Add Brogrammer Color Theme (Mac)

1. Download the [brogrammer.terminal](./assets/brogrammer.terminal) theme.
2. Go to _`Apple Terminal → Settings → Profiles → Circle with 3 dots → Import`_ to import the theme.
3. Select the theme from the list of themes.
4. Set the theme as the default profile by clicking on the _`Default`_ button at the bottom of the settings window.
5. Update the selection color for the profile to #45A2D2

## Install Oh My Zsh

1. Install Homebrew if you haven't already: [Homebrew Installation](https://brew.sh/)
2. Follow this guide: [How to install ZSH on macOS](https://github.com/ohmyzsh/ohmyzsh/wiki/Installing-ZSH#how-to-install-zsh-on-many-platforms)
