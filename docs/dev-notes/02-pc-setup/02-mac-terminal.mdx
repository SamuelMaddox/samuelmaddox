# Mac Terminal

![PowerLevel10k](./assets/powerlevel-10k.png)

## Add Brogrammer Color Theme (Mac)

1. Download the [brogrammer.terminal](./assets/brogrammer.terminal) theme.
2. Go to _`Apple Terminal → Preferences → Profiles → Gear Icon → Import`_ to import the theme.
3. Update the selection color for the profile to #45A2D2

## Install Oh My Zsh

Follow this guide: [How to install ZSH on macOS](https://github.com/ohmyzsh/ohmyzsh/wiki/Installing-ZSH#how-to-install-zsh-on-many-platforms)
